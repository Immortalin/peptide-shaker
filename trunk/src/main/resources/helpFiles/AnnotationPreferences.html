Spectrum Annotation

Here it is possible to set the settings used for the annotation of spectra.

The m/z accuracy will be used for the MS/MS annotation of the peaks.

If "Annotate Most Intense Peaks" is selected, only the peaks with intensities in the first quartile of all spectrum intensities will be annotated.

If the identification files were generated using SearchGUI, PeptideShaker will load the accuracy automatically!
