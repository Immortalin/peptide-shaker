Protein Filtering

Introduction

The protein filtering dialog makes it straightforward to filter the protein table in the Overview tab, e.g., to find proteins with a given post-translational modification, or to find all proteins with more than a certain number of spectra.



Filtering

When editing one of the filtering fields the protein table is automatically updated. When the wanted protein(s) have been located close the dialog by clicking the Exit button.



Case Sensitivity

Note that the Accession and Description fields are case sensitive, i.e., searching for 'Albumin' and 'albumin' is not the same thing.

Also note that both fields support regular expression search. For more details on regular expressions in Java see Java Regular Expressions.
