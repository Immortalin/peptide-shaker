Export Features

This dialog makes it straightforward to export data from PeptideShaker to tab separated text files at five levels:

  • Proteins
  • Peptides
  • PSMs
  • Search Engines
  • Complete Project

For the first four you can decide with features you want to export.

Note that you can only export features for one level at the time, i.e., proteins or peptides, not proteins and peptides. However, some of the export options includes information from other levels.

The export Project option is an advanced export and recommended for expert use only. In most cases you should be able to obtain the wanted export via the other export options.

Note that you can also export the content of most tables and all plots via the contextual export options available in the upper right corner of each table/plot.
