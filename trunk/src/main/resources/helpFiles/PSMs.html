The spectrum identification panel shows how peptides are matched to spectra using the various search engines.

Search Engine Performance

The search engine performance table displays how many spectra would be matched at 1% FDR using the search engines separately along with the corresponding Venn diagram.

Peptide-Spectrum Matches

For each spectrum it is possible to see all peptide assumptions made by the various search engines with the corresponding score and confidence. On top of these is displayed the PSM retained by PeptideShaker. Note that selecting the PSM of choice updates the display.
