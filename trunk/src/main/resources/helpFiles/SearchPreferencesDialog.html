Search Preferences

Introduction

In order to adapt the process to the search it is necessary to little information about it to PeptideShaker. Some are set by default, some need to be set manually. If the identification files were generated using SearchGUI, settings will be loaded automatically!

Search Parameters

The search parameters are mandatory for the import of identifications as well as for their processing and their display. Set here the enzyme used, the number of allowed missed cleavages and the fragment ion mass tolerance.

Modification Profile

The modification profile is a project-dependant set of expected modifications. PeptideShaker will look in this list when a PTM is found in the identification result. Modifications can be grouped into families and will then be processed together. It is vital that two different families do not contain PTMs of the same mass!

The modification profile can be saved and opened from here as well.

SearchGUI Parameters File

For more convenience, it is possible to load the standard settings from a SearchGUI file.
