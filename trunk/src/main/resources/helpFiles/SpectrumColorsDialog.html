Annotation Color

Here you can decide which colors to use for the peak annotations. To change a color simply click the colored box next to the ion type you would like to change the color for.

The change multiple ion type colors at the same time, first select multiple rows and the right click on one of the seleceted colors boxes.

Peak Color and Width

The color for the annotated and background peaks can be edited separatly. Click the colored boxes to edit the colors.

The width of the peaks can be also edited separatly for the annotated and background peaks.
