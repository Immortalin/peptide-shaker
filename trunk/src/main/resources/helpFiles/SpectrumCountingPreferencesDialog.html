MS/MS Quantification

PeptideShaker supports to types of MS/MS quantification methods:

  • emPAI: The exponentially modified protein abundance index is based on the number of validated peptides with respect to the theoretic number of peptides for a given protein. See Ishihama et al for more details.
  • NSAF: The normalized spectrum abundance factor is based on the number of validated spectra normalized by the protein size. See Paoletti et al for more details.

Switching between the two is done in this dialog. Here you can also decide if only validated hits are to be included in the methods.
