Introduction and Disclaimer

The Fractions tab shows you can overview of which fractions the proteins were detected in.

The values are estimations based on the confidence of a peptide/protein if found in a fraction alone in the context of the whole analysis.

These are not equal the confidence in the peptide/protein identifications when processing the fractions independently!

Independant fractions (like different donors, measurements) or replicates should be processed separately.

Plots

There are five plot options:
  • Peptides: the estimated number of peptides identified in each fraction.
  • Spectra: the estimated number of spectra identified in each fraction.
  • Intensities: the estimated average precursor intensity for each fraction.
  • Coverage: the estimated protein coverage for each fraction.
  • Molecular Weight: the estimated molecular weight for each fraction.

Selecting more than one protein in the table will display the values for all the selected proteins.

Note the use of the term 'estimated' above. This is not accidental. While the values can provide an idea of how the data ditributes across the fractions, further analysis will be required to be able to claim that this is indeed the case. However, an analysis that provides sound statistical ground for such claims is in development and will hopefully be available in the near future.
